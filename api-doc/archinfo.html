archinfo — Arch Info¶

archinfo is a collection of classes that contain architecture-specific information. It is useful for cross-architecture tools (such as pyvex).

class archinfo.arch.Arch(endness)¶

Bases: object

Represent an architecture. This class is responsible for architecture-specific things such as call emulation.

Variables:name – The name of the architecture.
bytes¶

Return the size in bytes of an int type as defined by the C standard.

int_bits¶

Return the size in bits of an int type as defined by the C standard.
